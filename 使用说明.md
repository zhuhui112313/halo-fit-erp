# HALO-FIT 外贸进销存系统 - 使用说明

**版本：** v0.1（测试版）
**日期：** 2026-02-27

---

## 📦 系统说明

本系统是基于 **Streamlit** 开发的本地单机版外贸进销存管理系统。

**特点：**
- ✅ 无需安装 Python（打包成 EXE）
- ✅ 双击直接运行
- ✅ 本地 SQLite 数据库（无需安装数据库）
- ✅ 简洁易用的界面
- ✅ 支持多币种

---

## 🚀 快速开始

### 方法 1：EXE 版（推荐）

1. 下载 `HALO-FIT 外贸进销存系统.exe`
2. 双击运行
3. 系统会自动打开浏览器
4. 使用默认账号登录：`admin` / `admin123`

### 方法 2：源码版

如果需要源码版本：

1. 安装 Python 3.8+
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行：
   ```bash
   streamlit run app.py
   ```

## 💾 数据备份

### 自动备份位置

系统会在 `backups` 文件夹中创建备份：
- 备份文件命名：`halo_fit_backup_YYYYMMDD_HHMMSS.db`
- 建议定期备份到外部存储

### 手动备份

1. 进入"系统设置"页面
2. 点击"立即备份"按钮
3. 下载生成的备份文件

### 恢复数据

1. 将备份文件替换 `halo_fit.db`
2. 重启系统

---

## ⚠️ 使用注意事项

### 重要提示

1. **数据安全**
   - 建议每天备份数据
   - 备份文件保存到外部存储
   - 避免误删数据库文件

2. **系统限制（v0.1 版本）**
   - 单机使用（不支持多人）
   - 本地数据（存储在本机）
   - 无权限系统（所有人相同权限）
   - 无微信登录
   - 无订单合并/拆分

3. **浏览器兼容性**
   - 推荐使用 Chrome / Edge / Firefox
   - 不支持 IE 浏览器

## 🐛 常见问题

### Q1: 双击 EXE 后没有反应？

**A:** 
1. 检查是否有杀毒软件拦截
2. 以管理员身份运行
3. 查看是否有错误提示

### Q2: 浏览器没有自动打开？

**A:**
1. 手动打开浏览器
2. 访问：`http://localhost:8501`

### Q3: 登录失败？

**A:**
1. 确认用户名和密码是否正确
2. 默认账号：admin / admin123
3. 删除 `halo_fit.db` 重新初始化

### Q4: 数据丢失怎么办？

**A:**
1. 检查 `backups` 文件夹是否有备份
2. 将备份文件复制并重命名为 `halo_fit.db`
3. 重启系统
